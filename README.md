# 🌾 AgriConnect — Farmer to Customer Marketplace

**Full Stack MERN Project**

AgriConnect is a full-stack web platform that enables farmers to sell agricultural produce directly to customers without intermediaries. The system improves market accessibility, pricing transparency, and user experience through a responsive web interface.

---

## 🚀 Features

* Farmer product listing and inventory management
* Customer browsing, search, and product filtering
* Secure user authentication and authorization
* Add to cart and order placement functionality
* RESTful API integration for data communication
* Responsive UI for mobile and desktop devices

## ⚙️ Installation & Setup

### Prerequisites

* Node.js
* MongoDB
* npm or yarn

### Steps

```bash
# Clone the repository
git clone https://github.com/yourusername/agriconnect.git

# Navigate to project folder
cd agriconnect

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install

# Run backend server
cd ../server
npm start

# Run frontend
cd ../client
npm start
```

Open browser → **http://localhost:3000**

## 📊 How It Works

1. Farmers register/login and add product details such as price, quantity, and description.
2. Customers browse available products and add items to cart.
3. Orders are processed via backend APIs and stored in MongoDB database.
4. Users can view order history and manage listings.
